🖥️ 需要启动服务器

多人联机需要运行 WebSocket 服务器，静态网页托管（如 GitHub Pages）无法支持。

本地运行方法：
1. 下载项目代码
2. 进入 games/mahjong-multiplayer 目录
3. 运行 npm install
4. 运行 npm start
5. 访问 http://localhost:3000

或者：部署到 Render / Railway 等支持 Node.js 的云平台
